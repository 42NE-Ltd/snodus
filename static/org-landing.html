Quick Start

# Clone and run with Docker
git clone https://github.com/42NE-Ltd/snodus
cd snodus
cp .env.example .env
# Edit .env: set ANTHROPIC_API_KEY and ADMIN_PASSWORD
docker compose up -d

# Create your first team, user, and API key
snodus teams create --name "My Team"
snodus users create --email you@company.com --name "Your Name" --team-id <TEAM_UUID> --role admin
snodus keys create --user-id <USER_UUID> --name dev

# Point your app to Snodus
export ANTHROPIC_BASE_URL=http://localhost:8080
export ANTHROPIC_AUTH_TOKEN=sk-...

What you get

Everything you need to manage AI API usage across your team.

Multi-Provider

Anthropic, OpenAI, and Ollama behind a single API. Automatic format translation between providers.

Smart Routing

Pass "model": "auto" and let Snodus pick Haiku, Sonnet, or Opus based on input complexity.

Virtual API Keys

Per-user keys with rate limiting, monthly budgets, rotation, and instant revocation.

Spend Tracking

Real-time cost tracking per model, user, and team. Dashboard with recent requests and breakdowns.

Budget Enforcement

Set monthly caps per key or team. Gateway returns 402 when the budget is exceeded.

Full CLI

Manage teams, users, keys, budgets, and providers from the command line.
